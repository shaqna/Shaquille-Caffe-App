# Shaquille-Caffe-App
Simple App for testing API created by PHP

## Software Development
- Hilt
- RESTful API
- Retrofit
- Kotlin
- Kotlin Coroutine, Kotlin Flow, StateFlow, SharedFlow
- MVVM with Clean Architecture
